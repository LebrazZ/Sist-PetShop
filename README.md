🐾 Sistema Pet Shop

Sistema desenvolvido para gerenciamento de um Pet Shop, permitindo o cadastro de clientes, pets, serviços e agendamentos de forma simples e intuitiva.

## 📋 Funcionalidades

- 👤 Cadastro de Clientes
- 🐶 Cadastro de Pets
- ✂️ Cadastro de Serviços
- 📅 Agendamento de Serviços
- 📊 Dashboard com quantidade de:
  - Clientes
  - Pets
  - Serviços
  - Agendamentos
- 🗑️ Exclusão de registros
- 💾 Armazenamento dos dados utilizando LocalStorage

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage

## 💻 Como executar

1. Clone o repositório

```bash
git clone https://github.com/LebrazZ/Sist-PetShop.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo **index.html** em seu navegador.

Não é necessário instalar dependências.
